# Brief de Key Visual

> **Cómo usar.** Copia este bloque y rellena cada `[así]`. Un KV sin brief se
> negocia con opiniones; con brief se juzga contra un criterio. Completa de arriba
> hacia abajo — si no puedes llenar "mensaje clave" en una frase, todavía no hay KV.
> Cierra apuntando al criterio de éxito, no al gusto. Para auditar el resultado usa
> `key-visual-audit-scorecard.md`.
