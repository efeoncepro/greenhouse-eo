# Swipe File — guía

> Un swipe file es una colección curada de copy excelente para **estudiar** (no copiar). Ver `SOURCES.md`.

## Qué guardar
- Headlines/subject lines que te hicieron parar.
- Leads/hooks que te "jalaron" a seguir leyendo.
- Sales pages/VSLs/emails que convierten (propios y de referencia).
- Taglines memorables; casos bien narrados; CTAs fuertes.
- Ejemplos de la voz de marca Efeonce que funcionan.

## Cómo usarlo (ético)
- **Estudia el patrón, no copies el texto.** Extrae la estructura/mecánica, aplícala con tu
  voice-of-customer y tu voz.
- Copiar copy ajeno = plagio + no encaja tu contexto. El swipe file entrena el ojo, no reemplaza el craft.

## Fuentes para nutrirlo
- El canon (`SOURCES.md`): Ogilvy, Caples, Halbert, Schwartz.
- Reviews/foros (voice-of-customer, `01`); newsletters/landings que admires.
- WebSearch/WebFetch para referencias vivas 2026.
