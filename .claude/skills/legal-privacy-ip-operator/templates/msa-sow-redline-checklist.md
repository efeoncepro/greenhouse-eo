# MSA / SOW — checklist de redline (advisory)

## Alcance
- [ ] Qué se entrega / qué NO (anti scope creep); SOW específico colgado del MSA
## Económico
- [ ] Precio, hitos, moneda, plazos de pago (net-N), reajuste/indexación (UF)
## IP (crítico — ver 06)
- [ ] Ownership de entregables (cesión de patrimoniales contra pago total)
- [ ] Assets/frameworks reusables de Efeonce RESERVADOS
- [ ] Derechos morales reconocidos (LATAM, irrenunciables)
- [ ] Derecho de portafolio pactado
## Riesgo
- [ ] Cap de responsabilidad + exclusión de daños indirectos
- [ ] Indemnidad recíproca y acotada
- [ ] Garantías razonables (no garantizar KPIs de negocio)
## Legales
- [ ] Ley aplicable + foro (cross-border consciente)
- [ ] Terminación + qué sobrevive
- [ ] DPA si se tratan datos del cliente (04)
- [ ] Cadena de IP con freelancers/vendors asegurada
- [ ] Firma electrónica (ZapSign adapter)
