# Sistema visual — Human / Agent / Interface

## Tesis visual

La web es la tercera presencia: una infraestructura compartida que traduce intención humana en acción agéntica autorizada y verificable.

## Gramática reusable para portadas del blog

Esta exploración prueba una dirección, no declara todavía un look universal. La metodología puede escalar si cada futura portada conserva la gramática y cambia la metáfora según la tesis del artículo.

| Capa | Invariante candidata | Qué puede cambiar |
|---|---|---|
| Concepto | Una relación intelectual visible, no decoración tecnológica | La metáfora específica |
| Jerarquía | Un protagonista y una tensión secundaria | Posición y escala |
| Composición | Mucho espacio negativo y significado dentro del centro crop-safe | Simetría, diagonal u oblicuidad |
| Materialidad | Táctil, sobria, con imperfección controlada | Papel, vidrio, piedra, metal, fotografía |
| Paleta | Neutro profundo + temperatura semántica + un acento funcional | Familias cromáticas del territorio |
| Tecnología | Se muestra como sistema o relación, nunca como cliché | Interfaz, protocolo, señal, evidencia |
| Marca | Reconocible por dirección antes que por logo | Firma y ubicación según superficie |

## Motivo de esta serie

El **intervalo gobernado**: una separación pequeña donde se hace visible una condición, permiso, confirmación o evidencia. No es una chispa ni un portal. En `WAG-V01` el intervalo vive alrededor de la ventana web central.

## Paleta funcional

- Blanco: campo editorial y espacio negativo dominante.
- Tinta `#022A4E` y azul profundo `#023C70`: estructura, profundidad y contraste.
- Azul activo `#0375DB`: comprensión agéntica y detalle mecánico.
- Naranja `#F55D01`, restringido: autoridad, confirmación y evidencia.
- Piel cálida natural: agencia humana; nunca naranja publicitario.

## Dirección de imagen

- Encuadre macro/editorial, como still life de museo contemporáneo.
- Luz lateral suave y localizada sobre un campo blanco limpio; sombras azul-gris muy contenidas.
- Profundidad moderada: interfaz e índices nítidos simultáneamente.
- La presencia agéntica es una mano robótica de diseño industrial: cerámica blanca, articulaciones mecánicas precisas y estructura azul Efeonce. Debe leerse como agente en menos de un segundo, sin convertirse en personaje humanoide.
- La ventana central es browser-inspired, no una réplica de Apple: barra superior mínima, controles abstractos, cero trade dress literal.
- UI sin texto legible. Su doble lectura se comunica mediante bloques visuales y una capa estructural alineada, nunca con código.

## Prohibiciones

- Negro como campo dominante, gradientes neón, halos, lens flare, bokeh decorativo o partículas abundantes.
- Azul/naranja cinematográfico usado como receta.
- Simetría muerta, centro vacío o interfaz secundaria.
- Robot humanoide, androide completo, guante militar, cerebro, red neuronal, nube de nodos.
- Logo/texto generado, letras residuales, watermark.
- Exceso de detalle que colapse en thumbnail.

## Estrategia de producción

1. Tres direcciones con una sola variable compositiva dominante.
2. Auditoría a miniatura y a resolución original.
3. Rechazo por semántica o anatomía; edición sólo para problemas localizados.
4. Iteración de una variable por ronda sobre la ganadora.
5. Composición determinística posterior para firma o UI exacta si resulta necesaria.
6. Derivados directos desde el master aprobado y QA de crops `16:9`, `1.905:1` y `1:1`.
