# Campaign visual system

## Concepto

**Alta frecuencia, estándar intacto.** Un colibrí singular permanece reconocible mientras el vuelo se expande en fases y formatos.

## Componentes

| Capa | Contrato |
| --- | --- |
| Sujeto | Un colibrí, no una bandada ni clones |
| Fondo | Navy profundo con gradación atmosférica controlada |
| Luz | Key light lateral; bordes iridiscentes; contraste alto |
| Color | Azul/verde de marca extendido por violeta iridiscente |
| Hook | Fases de ala o estela modular que nacen del mismo cuerpo |
| Tipo | Poppins ExtraBold/Bold, outlined con fontkit |
| Firma | Wordmark oficial blanco + `efeoncepro.com` |

## Fijo

Identidad del colibrí, fondo navy, iridiscencia, hook de frecuencia, tratamiento tipográfico, logo y URL.

## Flexible

Crop, escala del sujeto, posición del hook y copy según mensaje/formato.

## Prohibido

Texto generativo, más de un cuerpo completo, UI ficticia, factory literal, hologramas gratuitos, negro puro, logo redibujado, glow que destruya detalle o crop mecánico del master.
