## Reglas de uso

- Confirma la matriz y el parámetro antes de responder.
- Si hay más de un código para el mismo parámetro, no elijas una metodología por cuenta propia; informa la disponibilidad general y deriva la definición técnica.

- Para seguimiento de una muestra o informe existente, deriva al ingeniero responsable.
- Si no existe coincidencia exacta, no deduzcas que ANAM realiza el análisis.
