# ANAM attachment classification

This is a non-destructive Finder-friendly view of the original attachments. Files are linked to the sender-organized originals; they are not duplicate evidence or approved import packages.

- `01-hubspot-migration-candidates/`: source records that may be migrated only after the documented gates close.
- `02-hubspot-configuration-inputs/`: evidence for schema, forms, routing or reports; not record imports.
- `03-reference-only/`: contractual, historical or presentation context kept outside CRM.
- `04-exclude-from-hubspot/`: no useful CRM/configuration contract.
