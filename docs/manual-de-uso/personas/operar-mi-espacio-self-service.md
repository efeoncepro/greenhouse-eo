# Usar Mi Espacio self-service

> **Tipo de documento:** Manual de uso
> **Version:** 1.0
> **Creado:** 2026-06-15 por Codex
> **Modulo:** Mi Espacio / Self-service
> **Rutas:** `/my`, `/my/profile`, `/my/performance`, `/my/payment-profile`, `/my/payroll`, `/my/leave`, `/my/onboarding`, `/my/contractor`
> **Documentacion relacionada:** `docs/documentation/personas/my-space-self-service-end-to-end.md`

## Que puedes hacer

- Ver y revisar tu perfil.
- Revisar desempeño operacional propio.
- Descargar o revisar recibos de nomina propios.
- Solicitar permisos.
- Completar onboarding.
- Actualizar o solicitar perfil de pago.
- Si eres contractor, subir evidencia/boleta y revisar remittance.

## Revisar tu perfil

1. Abre `/my/profile`.
2. Revisa datos personales, organizacion, skills/tools y estado.
3. Si un dato legal o sensible esta mal, usa el flujo autorizado; no lo pidas por chat.

## Revisar performance

1. Abre `/my/performance`.
2. Selecciona periodo.
3. Lee el estado del periodo: parcial, cerrado, sin datos o degradado.
4. Usa la lectura de Nexa como guia operacional.
5. No lo trates como evaluacion formal HR.

## Actualizar perfil de pago

1. Abre `/my/payment-profile`.
2. Revisa el estado del perfil.
3. Crea o actualiza solicitud segun disponibilidad.
4. Espera aprobacion si el flujo la requiere.
5. No compartas datos bancarios completos por chat.

## Ver payroll propio

1. Abre `/my/payroll`.
2. Selecciona periodo.
3. Descarga o revisa recibo si esta disponible.
4. Si falta un recibo, puede depender del cierre/export de payroll.

## Contractor self-service

1. Abre `/my/contractor`.
2. Sube soporte/boleta/factura cuando corresponda.
3. Envia work submission.
4. Responde observaciones.
5. Recuerda: aprobar entrega no equivale a pago; el pago sigue en Finance.

## Que no hacer

- No pidas datos de otra persona desde Mi Espacio.
- No compartas RUT/documentos/cuentas completas en Nexa.
- No interpretes metricas parciales como cierre final.
- No uses Mi Espacio para corregir payroll de otra persona.
