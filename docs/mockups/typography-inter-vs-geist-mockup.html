Cómo leer este mockup

  • Toda la diferencia visual es entre body / form / table / chips / KPIs / dock / empty state.
  • Inter es la baseline implementada hoy (commit 5c4d84aa, deploy preview greenhouse-96gktkj39).
  • Geist se carga desde Google Fonts (Geist) directamente — no requiere registro local en este mockup.
  • Compará en particular: tablas densas (cómo se ve la columna ID y la columna monto), KPI hero (28-36px), pangrama de pesos (400-800), inputs y chips. Esos son los lugares donde la diferencia de personalidad se nota.
