# Sistema funcional de infografías editoriales

Las infografías editoriales de Efeonce son argumentos visuales gobernados, no ilustraciones decorativas. Cada
pieza existe para hacer visible un mecanismo, una frontera, una comparación, una ruta o una evidencia que el
texto no comunica con la misma velocidad.

## Qué entrega el sistema

- shell reconocible con composición semántica variable;
- source editable y delivery portable separados;
- variantes responsive/light-dark cuando el argumento lo necesita;
- footer de marca consistente en infografías de cuerpo;
- SVG directo para vectores cuando gana en seguridad, nitidez y peso;
- raster independiente para featured/OG/social;
- ALT, caption, fuentes, provenance y readback CMS;
- gates de autonomía, precisión, accesibilidad, SEO y shareability.

## Regla de marca

En una infografía de cuerpo, kicker, título y bajada viven arriba; toda la información de marca vive abajo. El
footer contiene fuente/fecha, wordmark oficial y sello `efeoncepro.com`. Portadas y OG son superficies distintas
y usan su propio contrato de crop, firma y formato.

## Estados

`brief → structure → production → qa → integrated → verified`. Un asset puede estar listo para el body y aún no
estar listo para redes. La indexación de Google es asíncrona y tampoco forma parte automática del estado
`verified`.
