# Research Briefs

## Purpose

This directory holds exploratory research that is **not yet committed to implementation**. Each brief captures a strategic direction, open questions, and what "ready for task" looks like.

## Lifecycle

```
Idea → Research Brief → Validated → Task(s) created → Brief archived
```

- **Research Brief**: Captures the opportunity, potential approaches, open questions, and validation criteria
- **Validated**: Open questions answered, approach selected, scope defined
- **Task(s) created**: One or more `TASK-###` entries in `docs/tasks/` with implementation specs
- **Archived**: Brief stays in this directory with status `archived` and links to resulting tasks

## Naming convention

```
RESEARCH-###-short-description.md
```
