# TASK-1534 — Voice-to-Creative-Prompt Motion

## Motion Contract

- Recording indicator uses tokenized restrained pulse plus visible timer/text.
- Start/stop changes immediately after confirmed browser state.
- Thought Capsule enters with small opacity/block reveal; no spatial flight or waveform theatre.
- Processing uses existing indeterminate status, never fake percentage.

## Reduced Motion

Pulse is removed; `Grabando`, timer, Stop and all terminal states remain visible. No meaning depends on movement.

## GVC / Micro Evidence

- Capture start, active, stop, processing, capsule reveal and discard desktop/mobile.
- Repeat active/ready with reduced effects; verify no layout shift hides Stop or source.
