# TASK-1485 — Globe Design System Visual Direction

## Decision

`Orbital Workshop`: una identidad de producción creativa propia, precisa y cinematográfica, construida desde
la shell Globe. Puede compartir colores de marca seleccionados con Efeonce/Greenhouse, pero no su sistema de
layouts, components, recipes, semantics o motion.

## Alternatives

1. **Greenhouse derivative:** consistente entre productos, rechazada por borrar autonomía y crear coupling.
2. **Blank-slate maximal:** distintiva, rechazada por costo/deriva sin use cases.
3. **Orbital Workshop — selected:** evolución incremental desde Globe, pattern por necesidad y evidencia.

## Principles

- Creative evidence over dashboard chrome.
- One dominant working plane; context and controls support it.
- Operational states are explicit, recoverable and audience-aware.
- Patterns enter as candidates and graduate only with real consumers.
- Greenhouse governs lifecycle/quality; Globe owns visual/interaction implementation.

## Initial evidence

Pattern Lab baseline futuro `globe.design-system.pattern-lab`, 1440×1000 y 390×844, con theme/state/a11y
fixtures. No baseline se promueve desde screenshots de Greenhouse.
