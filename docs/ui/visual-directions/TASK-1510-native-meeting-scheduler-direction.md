an abstract picker rather than a calendar. The operator rejected that direction. The current direction uses the
stable monthly-calendar + selected-day agenda mental model while preserving the portable renderer, Efeonce visual
language, accessible flow and receipt-gated measurement contract.

Historical rationale remains available in Git history; this file intentionally routes consumers to the active asset
so task/readiness tooling cannot treat two contradictory directions as selected.
