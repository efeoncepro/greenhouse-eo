# Objection Handling Map

> Toda venta es una conversación de objeciones. Anticípalas y neutralízalas en el copy. Ver `05`.

## Objeciones (del voice-of-customer, no inventadas)
| Objeción (en sus palabras) | Tipo | Respuesta en el copy | Prueba/mecanismo | Dónde va |
|---|---|---|---|---|
| "está caro" | precio | valor/ROI, comparación, framing | | oferta/FAQ |
| "no tengo tiempo" | esfuerzo | rapidez, "hecho por ti", pasos | | plan |
| "¿funcionará para mí?" | riesgo/fit | caso similar, segmentación | | prueba |
| "no confío" | confianza | garantía, autoridad, testimonio | | prueba/garantía |
| "¿por qué ahora?" | urgencia | consecuencia real de esperar | | cierre |
| | | | | |

## Reglas
- Objeciones **reales** (research, `01`), no las que tú imaginas.
- Neutraliza en el copy, no las escondas. Un testimonio que dice "yo también dudaba, pero…" vale oro.
- Cero manipulación: la respuesta es verdadera o cambia la oferta (`ANTIPATTERNS`).
