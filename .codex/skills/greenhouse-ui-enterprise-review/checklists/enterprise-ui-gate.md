# Enterprise UI Gate

- [ ] Design intent is written down.
- [ ] Screenshots exist for desktop, laptop, mobile.
- [ ] First fold has one dominant read path.
- [ ] Primary action has one obvious home.
- [ ] Mobile layout is intentional.
- [ ] Text does not clip or overlap.
- [ ] Empty/loading/error/degraded states are represented or consciously out of scope.
- [ ] Status is not color-only.
- [ ] Keyboard/focus path is preserved.
- [ ] Greenhouse primitives are reused where applicable.
- [ ] `DESIGN.md` remains the visual contract.
