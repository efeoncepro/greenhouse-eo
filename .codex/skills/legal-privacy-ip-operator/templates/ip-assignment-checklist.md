# Cesión de IP / derechos de uso — checklist (advisory)

## Si CEDES propiedad al cliente
- [ ] Cesión de derechos PATRIMONIALES explícita (no "work for hire" ciego en LATAM)
- [ ] Por medio / territorio / plazo / exclusividad definidos
- [ ] Condicionada a pago total
- [ ] Derechos morales reconocidos (irrenunciables, civil law)
- [ ] Assets reusables de Efeonce excluidos de la cesión

## Si LICENCIAS uso (o licencias un input de tercero)
- [ ] Scope / uso
- [ ] Territorio
- [ ] Plazo / término
- [ ] Medios / canales
- [ ] Exclusividad sí/no

## Inputs y releases
- [ ] Stock/música/fuentes/fotos/IA con licencia verificada (5 dimensiones)
- [ ] Model / property / talent / voice releases obtenidos
- [ ] IA: proveedor permite uso comercial + cesión (+ indemnidad ideal) (07)
